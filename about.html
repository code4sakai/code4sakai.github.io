Code for Sakaiとは

Code for Sakaiは、福井県坂井市で、「地域の課題をテクノロジーで解決する」シビックテック活動を行う団体に発展させようと思っているだけの個人のWebsiteです。
Code for Sakaiの管理人は、坂井市の地域の課題が何かを明らかにし、より良いまちを作れる環境を広めるためのシビックテック活動を行えたら良いなと漠然と思っています。
まずは、坂井市民の方はIssue機能を使ってご自分の考える課題を書き込んで頂ければ嬉しいです。

ミッション （理念、使命）、ビジョン （目標、未来像、ゴール）、バリュー（規範、行動基準）、事業

坂井市が将来にわたって存続できることを目指し、今住んでいる人にとって良いまちであるようにしたいと思っています。
当面はこの漠然とした思いを具体化していくことを目標としていきます。

Code for Japanや坂井市とは関係ありません

あくまで、非公式な活動です。
問題ございましたら、Issue等でご連絡をください。
